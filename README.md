# finalhomework 后端项目说明

本项目是使用 [Express.js](https://expressjs.com/) 框架搭建的 Node.js 后端项目，适用于前后端分离架构。

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run start
```

## 注意事项

* 请确保已安装 Node.js 版本 >= 14。
* 本项目为后端服务，不包含前端页面。
* 可根据实际业务拆分 controller 与 service 层。
